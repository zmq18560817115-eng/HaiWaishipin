# MVP 产出归档映射

验收通过的 `overseas-loc-mvp/runs/{slug}/` 可按以下规则复制到根目录：

| MVP 文件 | 目标目录 |
|----------|----------|
| `subtitles.srt` | `04_subtitles/` |
| `en-localization-pack.md` | `03_translation/` |
| `aspect-ratio-spec.md` | 本目录或项目子文件夹 |
| `broll/*.mp4` | `01_source_video/` 或 `06_editing/` |
| 完整 `runs/{slug}/` | `09_archive/{slug}/` |
