# 海外视频本地化工作流

本目录集中管理 **竞品对标 → 脚本出稿 → 交付剪辑** 的完整链路。

## 一键启动（推荐）

双击 **`启动工作台.cmd`** → 打开 **http://127.0.0.1:8788**

### 工作台五个页面

| 页面 | 作用 |
|------|------|
| **素材库** | 浏览 TikTok 竞品视频、查看 8 字段拆解、播放量 |
| **产品库** | 查看/编辑产品八大字段（卖点、痛点、禁用词等） |
| **脚本生成** | 左竞品 + 中产品 + 右模板 → 生成 → 完成交付 → 下载 zip |
| **成稿库** | 已交付项目列表与 zip 下载 |
| **反馈库** | 剪辑/运营反馈与投放数据 |
| **设置** | 一键同步 TikTok、拆解、模板、产品（无需命令行） |

> 8787 出稿页仍可用（兼容旧流程），但日常操作在 **8788 工作台** 即可完成。

## 交付 zip 内容（给剪辑 / 拍摄）

| 文件 | 说明 |
|------|------|
| `交付脚本包.md` | 七项脚本（标题 / 口播 / 分镜 / SeedDance 等） |
| `交付脚本包.json` | 同上，机器可读 |
| `subtitles.srt` | 英文字幕，PR / 剪映直接导入 |
| `剪辑单.html` | 5 镜剪辑对照（浏览器打开） |
| `broll/shot-*.mp4` | SeedDance 空镜（配置 `FAL_KEY` 后自动生成） |

## 应用目录

| 目录 | 端口 | 用途 |
|------|------|------|
| `海外视频本地化MVP/` | 8788 | 竞品库、AI 拆解、脚本生成、桥接到出稿 |
| `overseas-loc-mvp/` | 8787 | 英文本地化、字幕、交付 zip、SeedDance、成稿库 |
| `成稿库/` | — | 已交付项目索引（CSV + JSON） |
| `反馈库/` | — | 剪辑/运营反馈（Step 7） |

每条出稿项目保存在：`overseas-loc-mvp/runs/{slug}/`

试跑项目：`night-pumping-v1`、`flange-size-v1`；竞品桥接示例：`ref-001`

## 环境配置

1. 首次使用：双击 **`安装并检查开发环境.cmd`**（或仅检查用 **`检查开发环境.cmd`**）
2. 复制 `overseas-loc-mvp/.env.example` → `.env`
3. 填写 **`FAL_KEY`**（SeedDance 2.0，可选）
4. 知识库配置已统一到根目录 **`config/knowledge-sources.json`**（两应用共用 NAS 路径）

未填 `FAL_KEY` 时：Step 1–5 可完整跑通；Step 6 仅预览。

单独重启出稿页：双击 **`重启出海出稿.cmd`**

SeedDance 配置向导：双击 **`配置SeedDance.cmd`**

## 手工归档文件夹（可选）

| 文件夹 | 用途 |
| --- | --- |
| `00_inbox` | 临时接收新文件 |
| `01_source_video` | 原始视频 |
| `02_transcription` ~ `08_export` | 转写 / 翻译 / 字幕 / 配音 / 剪辑 / 审核 / 导出 |
| `09_archive` | 已完成项目归档 |
| `assets` | 字体、术语表 |
| `templates` | 归档映射模板 |

验收通过后可用：

```cmd
归档MVP产出.cmd night-pumping-v1
```

映射规则见 `templates/MVP产出归档映射.md`。

## 存储建议

- 大型视频不要放在 `temp/`
- 定期备份 `08_export`、`09_archive`
- 文件名建议：`项目名_语言_内容_版本_日期`
