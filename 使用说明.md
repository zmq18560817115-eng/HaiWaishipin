# 海外视频本地化工作台 - 使用说明

## 访问地址

**主工作台：** http://192.168.9.23:8788

在公司内网的任意电脑上，用浏览器（推荐 Chrome）打开即可使用，无需安装任何软件。

---

## 功能介绍

| 功能 | 说明 |
|------|------|
| 视频生成 | 选择产品 → 对标爆款 → AI 生成脚本 → 生成视频 |
| 爆款拆解 | 自动分析 TikTok 爆款视频的结构和卖点 |
| 素材库 | 竞品视频、产品资料、脚本快照管理 |
| 成稿反馈 | 已交付项目索引，审核/投放反馈闭环 |
| TikTok 采集 | 按关键词自动抓取 TikTok 公开视频元数据 |

---

## 快速开始

1. 打开 http://192.168.9.23:8788
2. 在「产品」中选择要推广的产品
3. 在「对标」中选择参考视频
4. 点击「开始创作」生成脚本和视频

---

## 注意事项

- 服务运行在公司内网服务器上，**24 小时运行**，随时可用
- 每人每天脚本生成上限 **10 条**（配额每天重置）
- 生成的视频和脚本会自动保存在服务器上
- 如遇到页面打不开，请确认电脑连接的是公司内网

---

## 常见问题

**Q: 页面打不开怎么办？**
A: 检查是否连接了公司内网，尝试刷新页面。如仍无法访问，请联系管理员。

**Q: AI 视频生成很慢？**
A: 每个分镜需要 1-2 分钟生成，一条完整视频大约需要 5-10 分钟，请耐心等待。

**Q: 可以同时多人使用吗？**
A: 可以，服务器支持多人同时访问。
