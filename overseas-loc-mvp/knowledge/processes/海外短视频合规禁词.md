# 海外 EN 禁词（命中须回 B4 改写）

| 禁止 EN | 改写方向 |
|---------|----------|
| medical grade | premium build quality 或删除 |
| pain-free / painless guarantee | comfortable sessions |
| increase milk supply | efficient pumping sessions |
| boost lactation | support pumping routine |
| completely silent | quieter for night sessions |
| cure / treat / diagnose | 删除 |
| best / #1 / guaranteed | 删除或 soften |
| FDA approved（无书面证据时） | 删除或合规书面批准后才可用 |

# 原则

- 不编造功效。
- 不生成真人哺乳画面。
- 不把 AI B-roll 当成功效证据。
- 成片中的 AI 片段必须保留来源与生成记录。
